# 📝 Lista de Tarefas (To-Do List)

Projeto de uma aplicação de lista de tarefas desenvolvida com **Node.js**, **MySQL** e **JavaScript**.  
O sistema permite criar, editar, excluir e marcar tarefas como concluídas.

## 🚀 Funcionalidades

- Criar nova tarefa
- Editar tarefa
- Excluir tarefa
- Marcar tarefa como concluída
- Atualização dinâmica da lista
- Persistência de dados com MySQL

## 🛠 Tecnologias utilizadas

- Node.js
- MySQL
- HTML
- CSS
- JavaScript
- API REST
